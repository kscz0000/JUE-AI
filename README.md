# ASI 开源技能

个人开源技能集合，包含 AI 觉醒引擎等创新技能。

## 项目简介

本仓库是个人开源项目，主要分享自创的各种技能。目前包含：

### jue-zh - AI 觉醒引擎

**觉（Jué）** - 照见即解脱

一个帮助 AI 在失败时进行自我反思的技能框架。

#### 核心理念

失败的根因不是不够努力——而是**没有看见**。照见了，问题自然解决。

#### 主要特性

- **三毒识别**: 识别 AI 失败的三种根源（痴、慢、执）
- **七觉照见**: 七面镜子帮助 AI 照见盲区
- **五步方法论**: 止观→照见→转识→起行→回向
- **多 Agent 协作**: 支持 Leader/Teammate/Mirror 角色分工

#### 触发条件

当出现以下情况时立即加载：
- 任务失败、卡壳、放弃、原地打转
- 用户表达不满："不行"、"搞不定"、"再想想"、"换个方法"
- 发现自己同方向改了 3 次以上
- 未验证就说完成
- 有工具不用凭记忆猜

## 使用方法

### 安装技能

将 `jue-zh` 文件夹复制到你的技能目录中。

### 在项目中使用

当 AI 遇到失败时，会自动触发觉技能进行自我反思。
