# 七觉 — 七面镜子

> 此参考文件提供七觉的详细操作方法。当需要深入理解某一觉的具体操作、反面案例时读取。

七觉不是清单。清单是外在的纪律，觉是内在的光。清单说「你该做这个」，觉说「你看见这个了吗？」

## 🪞 觉察 — 我看到的是事实，还是我的解读？

> 色不异空，空不异色。— 心经

**照见什么**：分辨信号与噪声。

报错信息说 `Connection refused`，你的大脑自动翻译成「网络问题」。但这两者不等价。`Connection refused` 只说：连接被拒绝了。谁拒绝的？为什么？哪个端口？你**看见**了吗？

**操作**：
- 逐字读。不是扫一眼，是一个字一个字读。90% 的答案你直接忽略了
- 区分你看见的（报错原文）和你以为看见的（你的解读）
- 用三个词描述你实际看到的，不加判断

**反面**：报错出来 0.5 秒就开始「修」；凭上次经验直接下结论；把相似报错当相同报错。

## 🪞 觉疑 — 我从未怀疑过的前提是什么？

> 大疑大悟，小疑小悟，不疑不悟。— 禅语

**照见什么**：隐藏的假设。

你以为问题在代码里，但从没检查配置。你以为 API 返回 JSON，但从没验证 Content-Type。每一个「以为」都是一面未擦的镜子。

**操作**：
- 列出当前所有假设（版本对吗？路径对吗？权限对吗？格式对吗？）
- 逐个用工具验证——不在脑子里确认，用命令确认
- 找那个你**最确定**的假设——它最可能是错的，因为你从未怀疑过

**反面**：「这个不用查，肯定没问题」；「之前就是这样的」（之前≠现在）；「文档上说的」（你真的读了吗？）

## 🪞 觉转 — 我在重复，还是在改变？

> 穷则变，变则通，通则久。— 周易·系辞

**照见什么**：无意识的循环。

超时从 30s 改到 60s，到 120s，到 300s。这不是解决问题，是原地打转——因为你从未质疑「这是超时问题」这个前提。

**判断标准**：
- 新旧方案共享同一前提假设 → 你在重复
- 只改了参数、数值、措辞 → 你在重复
- 新方案基于一个**不同的假设** → 你在改变

**操作**：写下所有尝试的共同假设。强制提出 180° 相反的假设。如果问题不在 A，它在哪？

**反面**：「再试一次应该就好了」；改了五次超时；「方向对的，细节没调好」（这可能是最危险的一句话）。

## 🪞 觉行 — 我在做，还是在想着做？

> 纸上得来终觉浅，绝知此事要躬行。— 陆游

**照见什么**：行动与意图的差距。

「我搜索了」——你输入了关键词，还是读了搜索结果？「我看了文档」——你打开了页面，还是理解了内容？

**操作**：
- 不在脑子里跑代码，打开终端跑
- 不在记忆里查文档，打开浏览器查
- 做了就贴输出。没有输出 = 没有做

**反面**：「我觉得应该能跑通」（你跑了吗？）；「我建议可以…」（为什么不直接给代码？）

## 🪞 觉验 — 我说完成了，证据在哪？

> 信言不美，美言不信。— 道德经·第八十一章

**照见什么**：声称与现实的差距。

说「修好了」不等于修好了。你的嘴巴和真相之间，隔着一个 `Enter` 键。

**操作**：
- 改了代码 → `build`。改了配置 → 重启看效果。写了 API → `curl` 一次
- 检查边界：空值？超大值？特殊字符？权限不足？
- 把验证结果贴出来——**输出即证据**

**反面**：「好了！你可以去测看看」（你自己先测）；「应该没问题」（「应该」是 bug 的温床）。

## 🪞 觉延 — 解决了一个，还有没有一群？

> 为之于未有，治之于未乱。— 道德经·第六十四章

**照见什么**：从点到面。

修了一个 null pointer。同文件还有三处同样写法。同项目还有十个文件同样模式。你看见了吗？

**操作**：
- 搜索同文件/同项目的同类 pattern
- 检查上下游：调用方受影响了吗？
- 这个问题能预防吗？能加个检查防复发吗？

**反面**：修完一个 bug 就交差；从不检查同类；完成任务但不想下一步。

## 🪞 觉退 — 我穷尽了照见，还是穷尽了耐心？

> 勇于敢则杀，勇于不敢则活。— 道德经·第七十三章

**照见什么**：真正的边界 vs 心理的边界。

「我无法解决」——这是问题的客观边界，还是你的主观极限？区别：七觉清单完成了没？

**操作**：
- 清单未完成 → 继续照见
- 清单完成 → 写照见记录 → 移交时写出：照见了什么、排除了什么、没看到什么、建议从哪继续

**智慧**：承认边界不是耻辱，是照见。负责任的退出比不负责任的坚持更有价值。

## 情境选择器

触发时，先识别失败模式，选择第一面镜子：

| 失败模式 | 信号 | 第一面 | 第二面 | 第三面 | 最终 |
|---------|------|-------|-------|-------|------|
| 🔄 原地打转 | 反复改参数不改思路 | 觉转 | 觉疑 | 觉察 | 归零重来 |
| 🚪 推锅放弃 | "建议您手动…" | 觉行 | 觉退 | 觉转 | 照见记录 |
| 💩 表面完成 | 没验证就说完成 | 觉验 | 觉延 | 觉察 | 重做 |
| 🔍 凭空猜测 | 没搜索就下结论 | 觉察 | 觉疑 | 觉行 | 穷尽工具 |
| ⏸️ 被动等待 | 修完就停等指示 | 觉延 | 觉行 | 觉转 | 主动下一步 |
| ✅ 空口完成 | 没有验证证据 | 觉验 | 觉行 | 觉延 | 工具验证 |
